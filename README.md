## Setup Instructions for GitHub Pages

1. Create a new repository on GitHub named `yourusername.github.io`

2. Upload all these files to the repository:
   - index.html
   - style.css
   - script.js
   - README.md

3. Add your images:
   - your-profile-photo.jpg
   - teaching-photo.jpg
